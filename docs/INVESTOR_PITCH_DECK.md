## The Solution

**Site to Spend** is an AI-powered platform that deploys and manages ad campaigns across Google, Facebook, Microsoft, and LinkedIn — autonomously.

No agency. No learning curve. No manual optimization.

You enter your website URL. Our AI extracts your brand, discovers your competitors, generates ad copy and creatives, deploys campaigns across platforms, and optimizes them 24/7 — all from a single dashboard.

### How It Works

1. **Brand Extraction (30–60 seconds)** — Enter your URL. Gemini Vision AI extracts your hex codes, fonts, imagery, and brand voice automatically. No questionnaires.

2. **Strategy & Creative Generation** — AI analyzes your business, discovers real competitors, and generates ad copy, images, and video tailored to your brand.

3. **Multi-Platform Deployment** — Campaigns deploy across Google Ads, Facebook, Microsoft, and LinkedIn simultaneously with platform-optimized settings.

4. **Autonomous Optimization** — Six AI agents run continuously — fixing broken ads, shifting budgets by time of day, A/B testing creatives, discovering competitors, and expanding audiences.

## Six Autonomous AI Agents

These agents run independently, 24/7, without human intervention.

| Agent | What It Does | Frequency |
|---|---|---|
| **Competitor Discovery** | Uses Google Search AI to find real competitors | Weekly |
| **Competitor Analysis** | Scrapes competitor websites for messaging, pricing, and value props; generates counter-strategies | Weekly |
| **Self-Healing** | Detects disapproved ads and auto-rewrites them while maintaining brand voice | Every 4 hours |
| **Budget Intelligence** | Dynamically shifts ad spend by time-of-day and day-of-week based on conversion patterns | Hourly |
| **Creative Intelligence** | Tracks A/B test performance at headline, description, and image level; generates new winning variations | Daily |
| **Audience Intelligence** | Manages Customer Match lists, segments audiences, and recommends lookalike expansion | On-demand |

These are not rule-based automations. Each agent uses Google's Gemini 3.x large language models to reason, analyze, and act with context about the specific business, its competitors, and its performance data.

## Technology

### Stack

| Layer | Technology |
|---|---|
| **Backend** | Laravel 12 (PHP 8.2+), PostgreSQL, Redis |
| **Frontend** | React 18, Inertia.js 2.0, Tailwind CSS |
| **AI** | Google Gemini 3.x (text, image, video via Veo 3.1) |
| **Ad Platforms** | Google Ads API v31, Meta Graph API v19, Microsoft Ads API, LinkedIn Marketing API |
| **Payments** | Stripe (Cashier subscriptions + ad spend billing) |
| **Queue** | Laravel Horizon (Redis-backed, 25+ job types) |
| **Infrastructure** | AWS (S3, compute), Laravel Forge |
| **Browser Automation** | Browsershot (brand extraction, competitor scraping) |

### Architecture Highlights

- **Management Account Pattern** — A single Spectra-owned management account per platform. Customer accounts are sub-accounts. No per-customer OAuth flows, no token management complexity, instant onboarding.
- **Event-Driven Agent System** — Agents are dispatched via queued jobs with retry policies, circuit breakers, and exponential backoff. Each agent operates independently and can be scaled horizontally.
- **Prepaid Ad Spend Billing** — 7-day credit system with daily reconciliation, auto-replenishment, and graduated failure handling (warn → reduce budget → pause campaigns).
